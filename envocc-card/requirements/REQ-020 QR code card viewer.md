# REQ-018 QR code card viewer
### Use case
- ดูบัตรออนไลน์ผ่าน QR code
### ผู้ใช้งานหลัก
- ประชาชนทั่วไปที่ต้องการดูบัตรพนักงานเจ้าหน้าที่
### Objectives
- เพื่อตรวจสอบและยืนยันบัตรพนักงานเจ้าหน้าที่
### Functional Requirements
1. ระบบจะต้องมีช่องทางการเข้าถึงบัตรพนักงานเจ้าหน้าที่หลังจาก scan QR code
2. ระบบจะต้องมีการกรอกรหัส qr code หลังจาก scan code
3. ระบบจะต้องมีการตรวจสอบ password ก่อนแสดงบัตร
4. ระบบจะต้องแสดงบัตรพนักงานเจ้าหน้าที่ในรูปแบบที่กำหนด
### Pre-conditions
- ก่อนที่จะแสดงบัตรผ่าน qr code
	- ระบบจะต้องมีข้อมูลของ qr code
	- ระบบจะต้องมีข้อมูลของบัตรพนักงานเจ้าหน้าที่ใบนั้น
	- บัตรพนักงานเจ้าหน้าที่จะต้องได้รับการอนุมัติ
### Post-conditions
- ถ้ามีการ scan qr code และกรอกรหัสถูกต้อง ระบบจะต้องแสดงบัตรในรูปแบบที่กำหนด
### Alternate conditions
- ไม่มี
### Exception conditions
- ถ้าหลังจาก scan qr code แล้วใส่รหัสไม่ถูกต้อง ระบบจะปฏิเสธการเข้าถึงบัตรพนักงานเจ้าหน้าที่
