# REQ-017 Online card viewer
### Use case
- ดูบัตรผ่านหน้าเว็บ
### ผู้ใช้งานหลัก
- user
### Objectives
- ดูบัตรที่ผ่านการอนุมัติแล้วทางหน้าเว็บ
### Functional Requirements
1. ระบบจะต้องมีช่องทางในการเข้าถึงการขอดูบัตร
2. ระบบจะต้องแสดงบัตรพนักงานเจ้าหน้าที่ในรูปแบบที่กำหนดไว้
### Pre-conditions
- ก่อนที่จะขอดูบัตรได้
	- user จะต้องเข้าสู่ระบบ
	- การร้องขอมีบัตรจะต้องได้รับการอนุมัติเรียบร้อยแล้ว
	- จะต้องมีข้อมูลของuser อยู่ในระบบ
### Post-conditions
- ถ้า user เข้าสู่ระบบ และร้องขอดูบัตรแล้ว ระบบจะต้องแสดงบัตรพนักงานเจ้าหน้าที่ตามรูปแบบที่กำหนดไว้
### Alternate conditions
- ไม่มี
### Exception conditions
- ไม่มี
