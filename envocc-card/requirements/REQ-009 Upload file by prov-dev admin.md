# REQ-009 Upload file by prov/dep admin
### Use case
- เพิ่มลายเซ็นของผู้ออกบัตรระดับ กอง หรือ จังหวัด
- เพิ่มตราประทับหน่วยงาน
### ผู้ใช้งานหลัก
- prov/dep admin
### Objectives
- เพื่อเพิ่มไฟล์ภาพของลายเซ็นของผู้มีอำนาจออกบัตร เพื่อนำไปประกอบกับการออกบัตรต่อไป
- เพื่อเพิ่มไฟล์รูปตราประทับของหน่วยงานระดับกอง หรือ จังหวัด เพื่อที่จะนำไปประกอบกับบัตรประจำตัวต่อไป
### Functional Requirements
1. ระบบจะต้องมีช่องทางให้ admin เข้าถึงการเพิ่มไฟล์
2. ระบบจะต้องมีการตรวจสอบข้อมูลของหน่วยงาน ก่อนที่จะเพิ่ม file
3. ระบบจะต้องมีที่กรอกข้อมูลของไฟล์นั้นๆ (metadata)
4. ระบบจะต้องให้ admin upload ไฟล์ลายเซ็นหรือไฟล์ตราประทับได้
5. ระบบจะต้องมีการตรวจสอบไฟล์ภาพก่อนบันทึกลงระบบ
### Pre-conditions
- ก่อนที่จะเพิ่มไฟล์ลงในระบบ
	- admin จะต้องเข้าสู่ระบบ
	- ระบบจะต้องมีข้อมูลของหน่วยงานอยู่ในฐานข้อมูล
### Post-conditions
- ถ้า admin 
	- กรอกข้อมูลถูกต้อง และ
	- Upload file  และ
	- file ถูกต้องตามรูปแบบ
- ระบบจะต้อง
	- บันทึก file ลงใน storage
	- บันทึกข้อมูลของรูปภาพ (metadata) ลงในฐานข้อมูล
### Alternate conditions
- ไม่มี
### Exception conditions
- ถ้า admin
	- กรอกข้อมูลไม่ถูกต้อง
	- ไม่ upload file
	- upload file ไม่ถูกต้องตามรูปแบบที่กำหนด
- ระบบจะต้อง
	- ปฏิเสธการเพิ่มไฟล์
	- แจ้งเตือนไปยัง admin
