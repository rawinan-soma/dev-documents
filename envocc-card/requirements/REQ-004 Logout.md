# REQ-004 Logout
### Use case
- ออกจากระบบ
### ผู้ใช้งานหลัก
- user
- admin ทั้งหมด
### Objectives
- เพื่อออกจากระบบและหยุดการใช้งาน
### Functional Requirements
1. ระบบจะต้องมีตัวเลือกให้ user ได้ออกจากระบบ
2. ระบบจะต้องมีการตรวจสอบว่าผู้ใช้งานเข้าสู่ระบบอยู่หรือไม่ ก่อนจะทำดำเนินการออกจากระบบ
### Pre-conditions
- ก่อนที่จะออกจากระบบ ผู้ใช้งานจะต้องทำการเข้าสู่ระบบก่อน
### Post-conditions
- ถ้าผู้ใช้งานร้องขอการออกจากระบบ ระบบจะต้องให้ผู้ใช้งานออกจากระบบ และไม่สามารถเข้าถึงทรัพยากรใดๆ ของระบบได้อีก
### Alternate conditions
- ไม่มี
### Exception conditions
- ไม่มี
